Pengenalan Konsep WEB dan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat serta menampilkan halaman website pada browser. HTML tidak termasuk ke dalam bahasa pemrograman karena tidak memiliki logika seperti variabel, percabangan, ataupun perulangan. Peran utama HTML adalah menyusun struktur halaman web dengan menggunakan berbagai tag yang menandai setiap bagian dari konten. Biasanya file HTML disimpan dengan ekstensi .html atau .htm dan dapat dibuka langsung menggunakan browser maupun dijalankan melalui web server.

Setiap dokumen HTML memiliki struktur dasar yang terdiri dari beberapa bagian penting yaitu DOCTYPE, <html>, <head>, dan <body>. DOCTYPE berfungsi untuk memberi informasi kepada browser bahwa dokumen tersebut menggunakan standar HTML. Bagian <head> berisi informasi mengenai halaman seperti judul, metadata, ataupun pengaturan lainnya. Sedangkan bagian <body> berisi seluruh isi utama halaman web seperti teks, gambar, tabel, maupun elemen lainnya yang akan ditampilkan kepada pengguna.

Dalam HTML terdapat beberapa komponen utama yaitu tag, elemen, dan atribut. Tag merupakan penanda yang digunakan untuk menentukan fungsi suatu bagian pada halaman web dan biasanya ditulis secara berpasangan seperti <p> dan </p>. Elemen adalah keseluruhan bagian yang terdiri dari tag pembuka, isi, serta tag penutup. Sementara itu atribut merupakan informasi tambahan yang dituliskan pada tag pembuka untuk mengatur tampilan atau perilaku elemen, contohnya atribut align, class, id, dan style.

HTML juga menyediakan berbagai cara untuk mengatur tampilan teks dan paragraf. Paragraf dibuat menggunakan tag <p> dan dapat diatur perataannya dengan atribut align seperti left, center, right, ataupun justify. Selain itu terdapat tag heading dari <h1> hingga <h6> yang digunakan untuk membuat judul dengan ukuran yang berbeda. Beberapa tag lain juga dapat digunakan untuk mengubah tampilan teks seperti <b> untuk teks tebal, <i> untuk teks miring, <u> untuk garis bawah, serta <strong> dan <em> untuk menandai teks yang dianggap penting.

Selain pemformatan teks, HTML juga memiliki beberapa fitur tambahan seperti atribut style, karakter khusus, dan komentar. Atribut style digunakan untuk mengatur tampilan elemen secara langsung, misalnya mengubah warna teks, ukuran huruf, ataupun warna latar belakang. HTML juga menyediakan berbagai karakter khusus seperti simbol matematika, simbol mata uang, dan tanda lainnya yang ditulis menggunakan kode tertentu. Selain itu terdapat komentar HTML yang ditulis dengan format <!-- komentar --> yang berfungsi untuk memberikan catatan pada kode tanpa ditampilkan pada halaman web.
